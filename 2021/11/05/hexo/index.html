hexo博客搭建

chendx 发布于 2021-11-05

安装:

官网下载git

https://git-scm.com/

官网下载nodejs

https://nodejs.org/zh-cn/

进入git

右键打开git bash here

node -v

查看nodejs版本看看下载好没有

npm -v

查看npm版本

因为国内有些慢所以先下载cnpm，命令行敲入

npm install -g cnpm –registry=https://registry.npm.taobao.org

安装完后cnpm -v查看一下，跳出一大段版本号就说明安装完成了

Hexo

cnpm安装hexo

然后查看hexo版本

然后看看路径，建一个文件夹blog

这里命令行创建，也可以手动创建，然后cd到blog路径下

然后初始化一个hexo博客

然后

hexo clean

清除缓存文件

hexo g

生成网站静态文件到默认设置的 public 文件夹

hexo s

启动本地服务器

然后浏览器进入localhost:4000，就会看到hexo自动升成的页面

注册github账户

https://github.com/

注册完后进入GitHub点击右上角的your repositories

New一个新仓库

仓库名要和你注册的名字一样，后面加上.github.io

生成SSH添加到GitHub

创建完回到git

git config –global user.name “yourname”

git config –global user.email “youremail”

查看一下

git config user.name

git config user.email

然后创建SSH,一路回车

ssh-keygen -t rsa -C “youremail”

已经生成了.ssh的文件夹。在你的电脑中找到这个文件夹。

id_rsa是你这台电脑的私人秘钥，不能给别人看的，id_rsa.pub是公共秘钥，可以随便给别人看，把这个公钥放在GitHub上

在git中，查看是否成功

ssh -T git@github.com

将hexo部署到GitHub

打开站点配置文件 _config.yml，翻到最后，修改为YourgithubName就是你的GitHub账户

需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub

cnpm install hexo-deployer-git –save

然后

hexo clean

hexo g

hexo d

注意hexo d时要你输入username和password。

部署成功后就可以在``http://yourname.github.io 看到你的博客了！！

设置个人域名

现在你的个人网站的地址是 yourname.github.io，如果需要设置个人域名，就得去腾讯云买一个域名。

注册一个腾讯云账户,我买的是 chendx.info，各个后缀的价格不太一样，比如最广泛的.com就比较贵。

你需要先去进行实名认证,然后在域名控制台中，看到你购买的域名。

点解析进去，添加解析。

点击快速添加解析，点击域名映射CNAME，添加你的 yourname.github.io

登录GitHub，进入之前创建的仓库，点击settings，设置Custom domain，输入你的域名chendx.info

然后在你的博客文件source中创建一个名为CNAME的txt文件，写上你的域名，然后重命名文件删除后缀。

在博客文件中的_config.yml文件中的url改成自己的域名

回到git，输入

hexo clean

hexo g

hexo d

在浏览器中输入你的域名就可以看到你的博客了。

hexo写文章

hexo new “a”

然后在你的博客文件source/_post中打开a.md文件就可以写文章了，当然不一定是a.md你想写什么名字就什么名字，你喜欢就好。

写完文章后，回到git，输入

hexo clean

hexo g

hexo d

这样你的文章就更新上去了。

hexo更改主题

进入hexo官网

https://hexo.io/

点击themes就可以看到很多主题了

看到喜欢的就点击底下的链接，进到github中clone到博客文件的themes中

git clone https://github.com/zoeingwingkei/frame.git themes/frame

在博客文件中的_config.yml文件中的拉到最后

theme改成主题的文件名

然后

hexo clean

hexo g

hexo d

这样主题就改好了
